# Frontend Mentor - Product Preview Card Component

This project is my solution for the Product Preview Card Component challengeon Frontend Mentor.  
 https://srenv.github.io/productPreviewCard/

## Overview

The goal of this challenge was to build a responsive product card component that displays a product image, description, price, and a call-to-action button.  
This challenge focuses on layout, typography, and responsive design principles.

## Features

- Fully responsive design
- Semantic HTML structure
- Clean layout with modern styling
- Hover effects on buttons and images (optional)

## Technologies Used

- HTML5
- CSS3 (Flexbox, media queries)
- Google Fonts

## What I Learned

- Better control of Flexbox for aligning card elements
- Responsive image handling and text wrapping
- Practice with hover effects and transitions
- Understanding spacing: margin vs padding
